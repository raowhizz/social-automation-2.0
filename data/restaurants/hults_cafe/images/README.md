# Images Folder

## Instructions

Place your dish images in this folder. The image filenames should match the "Image Filename" column in the `popular_items.xlsx` file.

### Example:
If your Excel has:
- `eggs_benedict.jpg` in the Image Filename column
- Then place the actual image file as `eggs_benedict.jpg` in this folder

### Supported Formats:
- JPEG (.jpg, .jpeg)
- PNG (.png)
- WebP (.webp)

### Image Guidelines:
- **Resolution**: Minimum 1080x1080px (square) or 1080x1350px (portrait)
- **Quality**: High-quality photos with good lighting
- **Composition**: Food should be the main focus
- **Background**: Clean, uncluttered backgrounds work best

### Naming Convention:
- Use lowercase with underscores: `dish_name.jpg`
- Be descriptive: `grilled_chicken_sandwich.jpg` (not `img001.jpg`)
- Match exactly with Excel filename (case-sensitive)
